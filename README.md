# Numpy_by_Kartik
# 📊 NumPy Learning Guide – By Kartik

## 📌 Overview
This project is a beginner-friendly walkthrough of the NumPy library in Python. It covers concepts from array creation to advanced operations like broadcasting and statistics.

---

## 🛠️ Requirements
- Python 3.x
- NumPy

Install:
pip install numpy

---

## 🚀 How to Run
python numpy_by_kartik.py

---

## 🧠 Learning Process

### 1. Array Creation
- Convert lists into NumPy arrays
- Create 1D, 2D, 3D arrays

### 2. Array Attributes
- shape, size, dtype, ndim

### 3. Initialization Methods
- zeros, ones, full, eye
- arange, linspace
- random arrays

### 4. Indexing & Slicing
- Access elements
- Slice arrays

### 5. Reshaping & Flattening
- reshape()
- flatten()

### 6. Stacking & Splitting
- vstack, hstack
- hsplit, vsplit

### 7. Math Operations
- +, -, *, /
- square, sqrt

### 8. Multiple Array Operations
- add, subtract, multiply, divide
- dot product
- transpose

### 9. Statistics
- sum, mean, median
- std, min, max

### 10. Comparison
- element-wise
- full array comparison

### 11. Broadcasting
- Operations on different shaped arrays

### 12. Handling NaN & Inf
- isnan()
- nan_to_num()

### 13. Save & Load
- np.save()
- np.load()
